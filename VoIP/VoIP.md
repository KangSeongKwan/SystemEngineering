# 1. 통신망
- 공간적으로 떨어져 있는 둘 이상의 사람이나 기계 사이를 결합하여 상호 통신이 가능하도록하는 시스템
- 구성도  
![image](https://user-images.githubusercontent.com/99636945/216515035-05812eaa-02db-4a66-a6a7-dbfef636db5d.png)
- 교환기 : 통신 신호를 받아서 원하는 곳으로 스위칭해주는 시스템
- 케이블 : 한 지점과 다른 지점을 연결해주는 물리적 매체(동축/광케이블)
- 전송 장치 : 두 지점간의 통신 신호를 묶어서 전달하는 장치

# 2. 교환
- 다수의 입력 단말과 다수의 출력 단말 사이를 임의 접속하는 기능이다.
- 통신 서비스 대상 단말이 많을 경우 설비 비용의 대부분을 차지하는 전송 매체 비용을 효과적으로 줄이기 위함
- 교환 기능 사용 전후 상황 예시는 다음과 같다.  
![image](https://user-images.githubusercontent.com/99636945/216518498-25752dfb-d899-4832-9ea1-7f222e005a5d.png)
- 위와 같이 교환 기능을 사용하면 전화기 4대의 통신 가능에 필요한 회선 수가 줄어듬을 알 수 있다.
- 교환을 활용한 통신 서비스에는 단말 장치, 전송 매체, 교환기가 필요하고 교환기는 연결 대상, 접속 경로 선택을 담당하는 핵심장비이다.

### 2-1. 디지털 교환기
- 반도체 스위치 소자가 갖는 빠른 동작속도를 이용한 교환기이다.
- 동작 원리로는 발신, 착신자가 실제적으로 물리적 연속성을 갖는 아날로그 교환기와는 다르게 아주 짧은 간격의 일정 주기마다 짧은 시간동안 유지되는 가상회선을 이용한다.  
![image](https://user-images.githubusercontent.com/99636945/216518865-09b0b4e2-0c3a-4286-974c-6e0b025ada65.png)
- 위 그림처럼 디지털 교환기는 초당 수백만번의 회전 동작을 하는 스위치를 이용하여 연결을 수행한다.

### 2-2. 교환기의 종류
- 교환기는 수동식, 기계식, 전자식 교환기로 구분된다.
- 수동식 교환기 : 회선 교환을 교환원이 직접 잭을 사용하여 서비스를 제공하는 교환기로, 자석식 교환기와 공전식 교환기가 있다.
- 기계식 교환기 : 가입자가 상대의 번호를 다이얼링하면 기계 장치가 자동적으로 가입자 상호간의 회선교환을 하는 방식, 스트로져, EMD, X-bar(크로스바) 교환기가 있다.
- 전자식 교환기 : 기계식의 많은 부분을 반도체로 대체하고, 컴퓨터를 내장시켜 컴퓨터에 의해 모든 교환회선을 제어하는 교환기로 반전자 교환기, 전전자 교환기가 있다.  
![image](https://user-images.githubusercontent.com/99636945/216519808-eedcaa09-dba8-471b-8e77-c373c1b0fcfd.png)
- 교환기는 기계식 -> 반전자식 -> 전자식 -> VoIP SoftSwitch로 발전되어 왔다.

# 3. 전송
### 3-1. 다중화
- 여러개의 신호를 규칙적으로 모으는 방법을 정의하는 것.
- 하나의 전송매체를 이용하여 2개 이상의 다른 정보를 동시에 전송할 수 있는 통신방식(다중 통신 방식이라고 함)
- 1개의 전송로에 다수의 신호를 중복시켜 회선 비용을 대폭 줄일 수 있다는 장점이 존재함. 전후 상황은 다음과 같다.  
![image](https://user-images.githubusercontent.com/99636945/216520054-d12b1132-dd54-434b-a1c1-11a76831c42f.png)
- 전송 다중화 방식에는 동기식 다중화, 비동기식 다중화가 있다.
- 프로세스 처리의 동기/비동기를 생각하면 된다.  
![image](https://user-images.githubusercontent.com/99636945/216520561-735fc459-3157-4ef7-a902-fd0948c12180.png)
- 위 그림에서는 N Step을 동기식 다중화, 1 Step을 비동기식 다중화라고 이해하면 된다.

### 3-2. 시분할 다중화 방식
- 전송 다중화 방식 중 시분할 다중화 방식이 존재한다.
- 주파수는 동일한 대역에서 전송하고 정보의 양에 따라 많은 시간이 필요하다.
- 디지털 통신 방식에서 많이 이용하고 전세계 디지털 통신의 다중화 방식으로 이용하고 있다.
- 125us 시간대를 가지는 각각의 신호가 시분할 다중화되는 원리

# 4. 교환망의 계위

### 4-2. 구성 요소의 특징
- 시내 교환국(End Office : EO) : 통상적인 일반 전화국이고 가입자가 연결되는 시내 교환국으로 수용 구역의 착발신의 호를 담당 
- 시외 교환국(Toll Center : TC) : 고속도로 톨게이트와 유사한 개념이고 시내 교환국 몇개를 집합한 시외 교환국이다. 자신의 관할 구역 내의 시내 교환기 상호간 및 타 시외 교환국과의 호 연결을 담당한다.
- 국제 관문국 : 각 시외국과 직접 연결되어 접수된 국제호를 처리
- 중계 교환기(Tandem Switch) : 교환기의 입출력 단자에 직접 가입자선을 수용하지 않고 둘 이상의 전화교환국에서의 중계선을 수용하며 그것을 상호간에 교환 접속하는 교환기이며 시내 중계 교환기, 시외 교환기 등이 있다.
